# Contributing

## Branch Workflow
- `main` is protected.
- Use `feat/<topic>` or `fix/<county>` branches.
- PRs require passing fixture tests.

## Commit Conventions
- `feat:` new scraper or feature
- `fix:` bug or selector patch
- `chore:` maintenance
- `docs:` documentation

## PR Template
1. What changed
2. Which county
3. Test evidence (screenshot or fixture)
4. Checklist:
   - [ ] Scraper works locally
   - [ ] Added fixtures
   - [ ] No creds in commit
